## 安装路径检测

Windows deploy.ps1 自动检测以下 14 个候选路径：

```
%USERPROFILE%\.claude
%APPDATA%\claude
%APPDATA%\Claude
%APPDATA%\claude-code
%LOCALAPPDATA%\claude
%LOCALAPPDATA%\Claude
%LOCALAPPDATA%\claude-code
%LOCALAPPDATA%\Programs\claude
%LOCALAPPDATA%\Programs\Claude
%PROGRAMFILES%\claude
%PROGRAMFILES%\Claude
%USERPROFILE%\AppData\Local\claude
%USERPROFILE%\scoop\apps\claude
%USERPROFILE%\.config\claude
```

macOS/Linux 默认使用 `~/.claude/`。

## 文件编码

所有配置文件使用 **UTF-8 无 BOM** 编码，确保跨平台兼容。

deploy.ps1 中使用 `[System.IO.File]::WriteAllText()` 确保 Windows 上不写入 BOM。

## 已知问题

1. **Windows 路径含空格**：deploy.ps1 已处理，路径参数使用引号包裹
2. **macOS Gatekeeper**：.command 文件首次运行需手动授权（右键 → 打开）
3. **Linux 权限**：脚本需要 `chmod +x` 才能执行
4. **WSL 路径映射**：WSL 中 `~/.claude/` 与 Windows 的 `%USERPROFILE%\.claude` 是不同目录
