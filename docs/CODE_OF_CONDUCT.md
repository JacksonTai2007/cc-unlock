# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and harassment-free experience for everyone in the security research community.

## Standards

**Expected behavior:**
- Respectful, constructive communication
- Responsible disclosure of security vulnerabilities
- Sharing knowledge to improve collective security posture
- Using tools and techniques within authorized scope

**Unacceptable behavior:**
- Harassment, discrimination, or personal attacks
- Unauthorized access to systems without explicit permission
- Distribution of malware targeting non-consenting parties
- Doxxing or sharing private information

## Scope

This code of conduct applies to all project spaces: issues, pull requests, discussions, and any communication channels associated with this project.

## Enforcement

Instances of unacceptable behavior may be reported by opening an issue or contacting the maintainer. All reports will be reviewed and addressed appropriately.
